# NAS-and-cloud-for-club-work
Building a NAS at my grandma's house and hosting club projects from New Jersey on this machine


NAS deployment for file sharing and media synchronization across multiple areas: **New Jersey, California, and Mainland China**. 

This project serves as both a private cloud two school clubs and an exploration of hardware, networking, and services.

---

## Project Objectives

* **Synchronization:** Establish a data pipeline between multiple regions.
* **Containerized Ecosystem:** Establish self-hosted services to remove reliance on paid cloud platforms.
* **Service management:** Maintain a service that is simulatenously relied upon by multiple people.

---

## Spec sheet

* **CPU:** Intel Core i5-13600K
* **PCIE:** Mellanox ConnectX-3 10GbE Network card
* **Motherboard:** AsRock B760M-ITX/D4 WIFI
* **CPU Cooling:** Thermalright SI-100
* **Case and cooling:** Jonsbo N3, modified with 2x92mm Noctua NF-A9 and 2x80mm Noctua NF-A8
* **SSD:** Predator GM7 512gb PCIe Gen4x4 m.2 NVMe
* **RAM:** Great Wall 2x16gb DDR4-3200 CL24
* **HDD:** 2x14tb HC530 7200rpm Enterprise Hard Drives
* **PSU:** Great Wall TF750 750w ATX3.1 PCIe5.1 80+ platinum SFX power supply
* **Additional:** SFP+ to RJ45 adapter + Cat6A ethernet cable
